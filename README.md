# ORI Course Recommender

Web app tư vấn khóa học TOEIC tự động hóa tương tác.

## Tính năng
- Trắc nghiệm 6 câu hỏi để phân loại học viên.
- Đề xuất khóa học phù hợp nhất dựa trên logic branching.
- Tính toán lộ trình và chi phí (trả góp/trọn gói).
- Gửi thông tin đăng ký về Google Sheet.

## Cài đặt
1. Clone repo
2. Chạy `npm install` (nếu cần)
3. Chạy `npx serve .` để test local

## Deploy
Sử dụng Vercel hoặc Netlify (Kéo thả folder hoặc kết nối GitHub).
